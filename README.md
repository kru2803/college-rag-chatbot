# EduRAG – AI-powered Education Knowledge Assistant

EduRAG is a Retrieval-Augmented Generation (RAG) chatbot that answers questions about students, teachers, and courses using a local Large Language Model.

The system retrieves relevant information from a knowledge base and generates responses using a locally running LLM.

🚀 Features
- Local AI chatbot powered by **Mistral (Ollama)**
- **Vector search with ChromaDB**
- **Sentence Transformers embeddings**
- Multi-document knowledge base
- **Streamlit web interface**
- Chat history support
- Retrieval-Augmented Generation (RAG)

  
🧠 Tech Stack
- Python
- Ollama (Mistral LLM)
- ChromaDB (Vector Database)
- Sentence Transformers
- Streamlit
